# Syntax

These are some of the Basic Syntaxes in Haskell

## Add
> x + y

this returns (x+y)

Eg) 5 + 6

11

## Subtract 

> x - y

this returns (x-y)

Eg) 6 - 5

1

## Multiply
> x * y

this returns (x*y)

Eg) 5 * 6

30

## Divide 
> x / y

this returns (x/y)

Eg)5/2

2.5

## Haskell follows BODMAS 

## Boolean Values
>True

True

>False

False

>not True

False

>False && True --"&&" is the "and" operator

False

>True || True -- "||" is the "or" operator

True
